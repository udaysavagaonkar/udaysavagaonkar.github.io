EnclaveAssertionAuthorityConfig

Configuration for an enclave assertion authority (i.e. generator or verifier).

Field Type Description
description AssertionDescription

Describes the target enclave assertion authority.
config bytes

A config to be used to initialize the authority. This field is an opaque string that may take on any format. It could, for example, be a serialized protobuf.
