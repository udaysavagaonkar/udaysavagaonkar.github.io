This package defines the messages for communicating with an Asylo enclave’s entry-points: initialization, (re-entrant) execution input and output, and finalization.

ServerConfig

Configuration passed to an enclave gRPC server during initialization.

Field Type Description
address string

The server’s address, which is either an IPv6 address or a Unix domain socket.
