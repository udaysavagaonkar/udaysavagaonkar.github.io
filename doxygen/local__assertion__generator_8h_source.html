30 /// An implementation of the EnclaveAssertionGenerator interface for SGX local
31 /// assertions.
32 ///
33 /// A LocalAssertionGenerator is capable of generating assertion offers and
34 /// assertions for SGX code identities that can be verified by SGX enclaves
35 /// running within the same local attestation domain.
36 class LocalAssertionGenerator final : public EnclaveAssertionGenerator {
37  public:
38  /// Constructs an uninitialized LocalAssertionGenerator.

40  /// The generator can be initialized via a call to Initialize().

66  private:
